# 项目简介 python83

本项目是基于Python语言的购物商城系统，采用Django框架进行开发。系统角色设计简洁明了，主要包括管理员与用户两大角色。

## 技术栈

- 后端：Python、Django框架
- 数据库：MySQL 5.7.33
- 前端：jQuery、Bootstrap、HTML、JSP
- 开发工具：PyCharm

## 功能模块

### 前台管理

- 热卖商品：展示热销商品信息
- 优惠资讯：提供最新的优惠信息
- 投诉建议：收集用户反馈，优化服务
- 购物车：管理用户选中的商品
- 收藏商品：允许用户收藏心仪商品
- 模拟支付：模拟真实的支付流程
- 合并下单：合并购物车中商品进行下单

### 后台管理

- 个人中心：管理个人信息
- 商品分类管理：维护商品分类信息
- 用户管理：管理用户账户信息
- 热卖商品管理：维护热卖商品信息
- 投诉建议管理：处理用户反馈
- 订单管理：管理用户的订单信息
- 系统管理：维护整个系统运行

## 系统角色

- 管理员：负责后台管理与维护
- 用户：在前台进行购物等操作

## 运行环境

- Python环境
- Django框架
- MySQL数据库

## 部署步骤

1. 安装Python及Django框架
2. 配置MySQL数据库
